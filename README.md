### 🌟 About Me 🌟
I am a passionate and motivated Full Stack Developer specializing in the MERN stack and modern web technologies. Currently pursuing my B.Tech in Computer Science, I combine strong academic foundations with hands-on project experience to build scalable, efficient, and user-centric applications. I am deeply interested in clean architecture, RESTful API development, system design, and data structures & algorithms.

Through my projects, I have worked with React.js, Node.js, Express, and MongoDB to build full-stack applications from the ground up. I have developed authentication systems, optimized database queries, and built intuitive user interfaces — all while maintaining clean and maintainable code. These experiences have sharpened my problem-solving skills and deepened my understanding of how to design efficient software systems.

I have built impactful projects including **HustleMap**, a full-stack job application tracker with status management and authentication, and **Rupee Roast**, a financial awareness platform that analyzes spending habits and provides smart financial insights. These projects reflect my ability to integrate frontend, backend, and database layers into cohesive and production-ready solutions.

My technical expertise includes C++, JavaScript, Python, React.js, Node.js, Express.js, MongoDB, MySQL, Git, and modern developer tools. I am comfortable working across the full development lifecycle — from planning and architecture to deployment and optimization.

I am continuously learning and growing in backend engineering, system design, and cloud technologies. I am actively seeking internship opportunities where I can contribute to meaningful projects, work in a collaborative engineering culture, and further strengthen my technical capabilities.
